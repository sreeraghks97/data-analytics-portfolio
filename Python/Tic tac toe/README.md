# Tic-Tac-Toe Game

## Project Overview

This project is a command-line implementation of the classic Tic-Tac-Toe game developed using Python.

The application allows two players to play alternately, validates user input, updates the game board after every move, detects winning conditions, and identifies draw scenarios.

---

## Features

- Two-player gameplay
- Interactive command-line interface
- Board updates after each move
- Input validation
- Win detection
- Draw detection
- Replay support (if implemented)

---

## Technologies Used

- Python

---

## Game Workflow

1. Display the game board.
2. Alternate turns between Player X and Player O.
3. Validate player input.
4. Update the board.
5. Check for a winner after each move.
6. Detect draw conditions.
7. End the game when a player wins or the game is drawn.
