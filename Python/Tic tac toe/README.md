# 🎮 Tic-Tac-Toe Game

A command-line implementation of the classic **Tic-Tac-Toe** game developed using Python. This project demonstrates core programming concepts by allowing two players to play interactively, validating user input, updating the game board after each move, and determining the game outcome based on winning or draw conditions.

---

## 📌 Project Overview

This project showcases the implementation of a two-player Tic-Tac-Toe game using Python. It focuses on applying fundamental programming concepts such as functions, loops, conditional statements, lists, and algorithm design to build an interactive console-based application.

---

## 🎯 Features

- Two-player gameplay (Player X vs. Player O)
- Interactive command-line interface
- Dynamic game board updates after each move
- User input validation
- Win detection across rows, columns, and diagonals
- Draw detection when no winning condition is met
- Replay support *(if implemented)*

---

## 🛠 Technologies Used

- Python

---

## 🎮 Game Workflow

1. Display the Tic-Tac-Toe board.
2. Allow players to take turns alternately.
3. Validate the selected position.
4. Update the game board with the player's move.
5. Check for a winning combination.
6. Detect a draw if all positions are filled.
7. Display the game result and end the session.
