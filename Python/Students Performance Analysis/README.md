# Student Performance Analysis

## Project Overview

This project analyzes student academic performance using Python, NumPy, Pandas, Matplotlib, and Seaborn to evaluate subject scores, attendance, departmental performance, and overall student achievement.

The analysis applies exploratory data analysis (EDA), statistical techniques, and visualization to identify performance trends and support data-driven academic decision-making.

---

## Business Problem

An educational institution wants to evaluate student performance across different departments to identify high-performing students, measure departmental performance, and understand the relationship between attendance and academic achievement.

The analysis helps answer questions such as:

- Which departments perform best academically?
- Who are the top-performing students?
- Does attendance influence academic performance?
- How are student scores distributed across departments?

---

## Dataset

The dataset contains student records including:

- Student ID
- Student Name

- Attendance
