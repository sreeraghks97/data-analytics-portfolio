# Sales Data Analysis

## Project Overview

This project analyzes sales transaction data using Python, NumPy, Pandas, Matplotlib, and Seaborn to evaluate business performance across products, regions, and time.

The analysis focuses on identifying sales trends, regional performance, product demand, pricing patterns, and revenue distribution through exploratory data analysis (EDA) and data visualization.

---

## Business Problem

A retail organization wants to evaluate its sales performance to answer key business questions, including:

- Which regions generate the highest revenue?
- Which products sell the most?
- How does revenue change throughout the year?
- Is there a relationship between product pricing and revenue?
- Which business metrics are most strongly correlated?

The insights can support inventory planning, pricing strategies, and regional sales optimization.

---

## Dataset

The dataset contains **500 sales transactions** with the following attributes:

- Date
- Region
- Product
- Units Sold
- Unit Price
- Revenue

Revenue is calculated as:

```
Revenue = Units Sold × Unit Price
```
