# E-Commerce Customer Purchase Analysis

## Project Overview

This project analyzes an e-commerce transaction dataset to explore customer purchasing behavior, spending patterns, payment preferences, and customer demographics using Python and Pandas.

The objective is to perform exploratory data analysis (EDA) and extract business insights that can support customer segmentation and business decision-making.

---

## Business Problem

An e-commerce company wants to better understand its customers by analyzing transaction data to answer questions related to purchasing behavior, customer demographics, payment methods, and spending patterns.

---

## Dataset

The dataset contains **10,000 customer transactions** with information including:

- Customer details
- Purchase price
- Payment method
- Credit card information
- Email provider
- Occupation
- Language
- Purchase time (AM/PM)

The dataset used in this project is for learning purposes.

## Analysis Performed

### Customer Spending Analysis

### Customer Demographics

- Language preferences
- Occupation analysis
- Most common job titles

### Purchase Behavior

- AM vs PM purchase activity
- High-value customer identification

### Payment Analysis

- Credit card provider analysis
- Card expiration analysis

### Customer Information Analysis

- Email provider distribution
- Transaction lookup using unique identifiers

## Key Insights

- Calculated average customer spending per transaction.
- Identified the most common customer occupations.
- Compared purchasing activity during AM and PM.
- Analyzed customer payment preferences.
- Identified the most popular email providers.
- Retrieved customer-specific transaction information using unique identifiers.
