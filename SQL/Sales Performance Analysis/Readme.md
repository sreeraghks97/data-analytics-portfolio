# Sales Performance Analysis

## Project Overview

This project analyzes sales transaction data to evaluate overall business performance by calculating total revenue, product-wise sales, and customer-wise revenue.

Using SQL, sales records are aggregated to generate insights that support product performance evaluation and customer analysis.

---

## Business Problem

Businesses need to understand how their products and customers contribute to overall revenue.

This analysis aims to answer:

- What is the total revenue generated?
- Which products contribute the most to sales?
- Which customers generate the highest revenue?

---

## Dataset Description

The dataset contains sales transaction records.

| Column | Description |
|---|---|
| CustomerID | Unique customer identifier |
| ProductID | Product identifier |
| PurchaseDate | Date of purchase |
| Quantity | Quantity purchased |
| Revenue | Revenue generated from the transaction |

---

## Analysis Performed

- Calculated total business revenue.
- Analyzed product-wise sales and revenue.
- Identified top customers based on revenue.
- Summarized sales performance using SQL aggregation.

---

## Business Questions Answered

1. What is the total revenue generated?

## Key Insights

- Total revenue provides an overview of business performance.
- Product-level analysis highlights top-selling products.
