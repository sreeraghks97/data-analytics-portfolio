# Customer Data Cleaning

## Project Overview

This project focuses on cleaning and standardizing customer data by correcting inconsistent name formats.

Using SQL, customer names with incorrect capitalization are transformed into a consistent format to improve data quality and usability.

---

## Business Problem

Customer databases often contain inconsistent data due to manual entry errors, such as incorrect capitalization or formatting issues.

This analysis aims to:

- Identify inconsistent customer names.
- Standardize name formatting.
- Improve data accuracy for reporting and analysis.

---

## Dataset Description

The dataset contains customer name records.

| Column | Description |
|---|---|
| Name | Customer name |

---

## Analysis Performed

- Identified inconsistent name formatting.
- Applied string functions to standardize customer names.
- Updated records with corrected formatting.

---

## Key Insights

## Business Recommendations

- Regularly clean and standardize customer records.
- Maintain consistent formatting standards across databases.
