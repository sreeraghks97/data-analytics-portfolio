# Unemployment Data Analysis (Power BI)

## 📌 Overview
Interactive Power BI dashboard analyzing unemployment statistics segmented by age, gender, and year.

## 📂 Dataset
- Source: Unemployment dataset
- Format: CSV/Excel with demographic and year-wise unemployment data

## ⚙️ Techniques Used
- Power BI data modeling
- DAX measures for KPIs
- Interactive filters and slicers

## 📊 Key Insights
- Men aged 25–34 had the highest unemployment.
- Peak unemployment occurred in 2010, with a sharp decline by 2015.
- Overall, men had higher unemployment numbers compared to women.
