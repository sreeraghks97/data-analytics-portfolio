# 📊 Unemployment Data Analysis | Power BI

![Dashboard Preview](Unemployment%20Data%20Analysis%20Dashboard.png)

## 📌 Project Overview

This Power BI dashboard analyzes unemployment statistics to explore workforce trends across different age groups, genders, and years. The interactive dashboard enables users to examine demographic patterns and understand how unemployment has changed over time.

---

## 🎯 Business Objective

The objective of this project is to analyze unemployment data to identify demographic trends, compare unemployment levels across different age groups and genders, and examine changes over time. The dashboard supports data-driven analysis through interactive visualizations and dynamic filtering.

---

## 📂 Dataset

**Source:** Unemployment Dataset

The dataset contains unemployment-related information, including:

- Year
- Age Group
- Gender
- Unemployment Statistics
- Demographic Details

---

## 📊 Dashboard Features

- Interactive filters for demographic analysis
- Year-wise unemployment trend analysis
- Age group comparison
- Gender-based unemployment analysis
- Dynamic visualizations for interactive data exploration

---

## 🛠️ Tools & Skills Used

- Power BI
- Data Cleaning
- Data Modeling
- DAX
- Interactive Dashboards
- Data Visualization
- Slicers & Filters

---

## 🔍 Key Findings

- Men aged **25–34** recorded the highest unemployment levels in the dataset.
- Unemployment peaked around **2010** before showing a noticeable decline by **2015**.
- Overall, male unemployment figures were higher than female unemployment figures.
- Interactive filtering enables detailed comparison of unemployment trends across different demographic groups and years.

---

## ✅ Conclusion

This dashboard transforms unemployment statistics into an interactive analytical solution that highlights demographic trends and changes over time. It demonstrates the use of Power BI for data modeling, DAX calculations, and dashboard development while enabling users to explore unemployment patterns through dynamic visualizations.
