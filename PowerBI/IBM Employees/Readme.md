# 👨‍💼 IBM Employee Data Analysis | Power BI

![Dashboard Preview](IBM%20Employee%20Data%20Analysis.png)

## 📌 Project Overview

This Power BI dashboard analyzes IBM employee data to provide insights into workforce demographics, departmental distribution, education background, and employee attrition. The interactive dashboard enables users to explore workforce composition and identify patterns across different employee segments.

---

## 🎯 Business Objective

The objective of this project is to analyze employee data to understand workforce distribution, demographic trends, and attrition patterns. The dashboard supports HR analytics by providing interactive visualizations that help explore organizational structure and employee characteristics.

---

## 📂 Dataset

**Source:** IBM HR Analytics Dataset

The dataset contains employee-related information, including:

- Department
- Gender
- Education Field
- Job Role
- Age
- Attrition
- Employee Demographics

---

## 📊 Dashboard Features

- Interactive filters for workforce analysis
- Department-wise employee distribution
- Gender-based workforce analysis
- Education field distribution
- Employee attrition analysis
- Dynamic visualizations for interactive exploration

---

## 🛠️ Tools & Skills Used

- Power BI
- Data Cleaning
- Data Modeling
- DAX
- Interactive Dashboards
- Data Visualization
- Slicers & Filters

---

## 🔍 Key Findings

- Research & Development has the highest number of employees.
- Male employees slightly outnumber female employees.
- Life Sciences and Medical are the most common education fields among employees.
- Interactive filtering enables detailed exploration of workforce demographics and organizational distribution.

---

## ✅ Conclusion

This dashboard transforms employee data into an interactive HR analytics solution that provides insights into workforce composition and organizational trends. It demonstrates the effective use of Power BI for data modeling, DAX calculations, and dashboard development while enabling users to analyze employee demographics and departmental distribution through dynamic visualizations.
