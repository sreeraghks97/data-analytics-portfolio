# 🌍 Global Terrorism Analysis | Power BI

![Dashboard Preview](GlobalTerrorism.png)

## 📌 Project Overview

This Power BI dashboard analyzes global terrorism incident data to identify trends in attack frequency, target types, weapon usage, terrorist organizations, and geographical distribution. The interactive dashboard enables users to explore terrorism patterns across different regions and time periods.

---

## 🎯 Business Objective

The objective of this project is to analyze global terrorism data to uncover trends in terrorist activities, identify high-risk regions, and examine attack characteristics. The dashboard provides an interactive platform for exploring historical incident data and understanding global terrorism patterns.

---

## 📂 Dataset

**Source:** Global Terrorism Database (GTD)

The dataset contains historical terrorism incident records, including:

- Year of Incident
- Country and Region
- Terrorist Group
- Attack Type
- Target Type
- Weapon Type
- Casualties
- Success of Attack

---

## 📊 Dashboard Features

- Interactive filters for exploring terrorism incidents
- Region-wise and country-wise analysis
- Terrorist group and attack type distribution
- Weapon type analysis
- Trend analysis of incidents over time
- Dynamic visualizations for interactive exploration

---

## 🛠️ Tools & Skills Used

- Power BI
- Data Cleaning
- Data Modeling
- DAX
- Interactive Dashboards
- Data Visualization
- Map Visualizations
- Slicers & Filters

---

## 🔍 Key Findings

- Taliban and ISIL were responsible for the highest number of deadly incidents in the dataset.
- Explosives were the most frequently used weapon type.
- Terrorist incidents peaked around 2014.
- Interactive filtering enables detailed analysis of attack patterns across regions, terrorist groups, and weapon types.

---

## ✅ Conclusion

This dashboard transforms historical terrorism data into an interactive analytical tool for exploring global attack patterns and regional trends. It demonstrates the use of Power BI for data modeling, DAX calculations, and geospatial visualization while enabling users to investigate terrorism incidents from multiple analytical perspectives.
