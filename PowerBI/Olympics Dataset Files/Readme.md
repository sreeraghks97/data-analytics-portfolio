# Olympics Analysis (Power BI)

## 📌 Overview
Interactive Power BI dashboard analyzing Olympic medal counts and participation trends.

## 📂 Dataset
- Source: Olympics dataset
- Format: CSV/Excel with medal and athlete records

## ⚙️ Techniques Used
- Power BI data modeling
- DAX measures for medal counts
- Treemaps, pie charts, and line charts

## 📊 Key Insights
- USA led overall medal counts.
- Male athletes contributed the majority of medals.
- Athletics, Gymnastics, and Swimming were top sports.
