# 🏅 Olympics Analysis | Power BI

![Dashboard Preview](olympics%20data%20dashboard.png)

## 📌 Project Overview

This Power BI dashboard analyzes historical Olympic data to explore medal distribution, athlete participation, and country-wise performance. The interactive dashboard enables users to examine medal trends, compare countries, and gain insights into the performance of different sports across Olympic events.

---

## 🎯 Business Objective

The objective of this project is to analyze Olympic data to identify medal-winning trends, evaluate country performance, and understand athlete participation across different sports. The dashboard provides an interactive platform for exploring historical Olympic achievements through dynamic visualizations.

---

## 📂 Dataset

**Source:** Olympics Dataset

The dataset contains historical Olympic records, including:

- Country
- Athlete
- Gender
- Sport
- Event
- Medal Type
- Olympic Year

---

## 📊 Dashboard Features

- Interactive filters for Olympic data exploration
- Country-wise medal analysis
- Sport-wise medal distribution
- Gender-based medal comparison
- Medal trend analysis across Olympic years
- Dynamic visualizations for interactive analysis

---

## 🛠️ Tools & Skills Used

- Power BI
- Data Cleaning
- Data Modeling
- DAX
- Interactive Dashboards
- Data Visualization
- Treemaps
- Slicers & Filters

---

## 🔍 Key Findings

- The United States secured the highest overall medal count in the dataset.
- Male athletes contributed the majority of medals.
- Athletics, Gymnastics, and Swimming were among the most successful sports.
- Interactive filtering allows users to compare medal performance across countries, sports, and Olympic years.

---

## ✅ Conclusion

This dashboard transforms historical Olympic data into an interactive analytical solution that highlights medal distribution, athlete participation, and country performance. It demonstrates the use of Power BI for data modeling, DAX calculations, and interactive dashboard design while enabling users to explore Olympic trends from multiple perspectives.
