# Sales & Inventory Analysis (Power BI)

## 📌 Overview
Dashboard analyzing product sales, discounts, and customer purchasing behavior.

## 📂 Dataset
- Source: Sales and inventory dataset
- Format: CSV/Excel with product and transaction records

## ⚙️ Techniques Used
- Power BI data modeling
- DAX measures for sales KPIs
- Bar charts, scatter plots, and donut charts

## 📊 Key Insights
- Asparagus and Rhubarb were top-selling products.
- Strawberry generated the highest sales revenue.
- Carrot and Rhubarb had the highest quantities sold.
