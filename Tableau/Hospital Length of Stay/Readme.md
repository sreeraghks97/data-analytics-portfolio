# 🏥 Hospital Length of Stay (LOS) Dashboard

An interactive Tableau dashboard developed to analyze Hospital Length of Stay (LOS) data. The dashboard visualizes patient demographics, admission categories, and departmental performance to support healthcare analytics through interactive and business-focused visualizations.

---

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on hospital Length of Stay (LOS) data using Tableau. The dashboard explores patient demographics, admission categories, severity levels, and departmental performance to uncover patterns in hospitalization and present insights through an interactive dashboard.

---

## 🎯 Objectives

- Analyze average Length of Stay (LOS) across hospital departments.
- Explore patient demographics and hospitalization trends.
- Compare LOS across admission categories and severity levels.
- Build an interactive dashboard for efficient data exploration.
- Present healthcare insights using intuitive visualizations.

---

## 📂 Dataset

**Dataset:** Hospital Length of Stay (LOS) Records

The dataset includes:

- Patient ID
- Age
- Gender
- Department
- Admission Category
- Severity Level
- Length of Stay (LOS)

## 📊 Dashboard Components

- **Department LOS Comparison** – Compare average Length of Stay across hospital departments.
- **Patient Age Analysis** – Analyze LOS across different age groups.
- **Severity Level Analysis** – Explore hospitalization trends based on patient severity.
- **Admission Category Analysis** – Compare patient admissions across different categories.
- **Interactive Filters** – Enable dynamic exploration of the dashboard.
