# ✈️ Air Travel Analysis Dashboard

An interactive Tableau dashboard developed to analyze air travel data, passenger trends, flight activity, and route performance. The dashboard enables users to explore travel patterns, route popularity, and operational metrics through interactive visualizations, supporting data-driven airline and travel analysis.

---

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on air travel data using Tableau. The dashboard explores flight activity, passenger demand, route performance, and travel trends to identify patterns and present operational insights through interactive visualizations.

---

## 🎯 Objectives

- Analyze flight activity across routes and time periods.
- Explore passenger travel trends and seasonal demand.
- Compare route performance using flight volume and distance metrics.
- Build an interactive dashboard for airline data analysis.
- Present operational insights using effective data visualization.

---

## 📂 Dataset

**Dataset:** Flight Records

The dataset includes:

- Flight ID
- Origin
- Destination
- Date
- Passenger Count
- Distance

## 📊 Dashboard Components

- **Route Analysis** – Visualize flight activity across different routes using maps.
- **Passenger Trend Analysis** – Explore travel demand over different time periods.
- **Flight Activity Analysis** – Monitor daily and route-wise flight performance.
- **Distance Analysis** – Compare flight distances across routes.
- **Interactive Filters** – Enable dynamic exploration of airline data.
