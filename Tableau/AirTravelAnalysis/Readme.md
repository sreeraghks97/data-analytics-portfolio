# Air Travel Analysis (Tableau)

## 📌 Overview
Dashboard visualizing airline performance and passenger trends.

## 📂 Dataset
- Source: Flight dataset
- Format: CSV/Excel with route and passenger data

## ⚙️ Techniques Used
- Tableau data connections
- Map visuals for routes
- KPIs for flight counts and distances

## 📊 Key Insights
- Seasonal peaks in travel demand.
- SFO–LAX was the busiest route.
- March 6, 2020 had the highest daily flights.
