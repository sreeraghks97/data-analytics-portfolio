# HR Analysis (Tableau)

## 📌 Overview
Dashboard analyzing employee attrition, demographics, and job roles.

## 📂 Dataset
- Source: HR dataset
- Format: CSV/Excel with employee records

## ⚙️ Techniques Used
- Bar charts for attrition by role
- Scatter plots for promotion vs attrition
- KPIs for age and income groups

## 📊 Key Insights
- Sales department had the highest attrition.
- Younger employees more likely to leave.
- Income group strongly influenced attrition rates.
