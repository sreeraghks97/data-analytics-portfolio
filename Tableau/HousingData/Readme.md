# House Data Analytics (Tableau)

## 📌 Overview
Dashboard analyzing house prices against crime rate, pollution, distance, and rooms.

## 📂 Dataset
- Source: Housing dataset
- Format: CSV/Excel with property and environmental data

## ⚙️ Techniques Used
- Scatter plots for correlation analysis
- Filters for crime rate, pollution, and distance
- KPIs for average house price

## 📊 Key Insights
- Crime rate and pollution had strong negative correlation with house prices.
- Number of rooms positively correlated with price.
- Distance from city center influenced affordability.
