# Rainfall Data Analysis (Tableau)

## 📌 Overview
Dashboard analyzing rainfall patterns across Indian states and seasons.

## 📂 Dataset
- Source: Rainfall dataset
- Format: CSV/Excel with state and district rainfall data

## ⚙️ Techniques Used
- Bar charts for seasonal rainfall
- Maps for district distribution
- KPIs for monsoon contribution

## 📊 Key Insights
- Monsoon contributed over 90% of annual rainfall in several states.
- Uttar Pradesh recorded the highest annual rainfall.
- Seasonal variation was significant across regions.
