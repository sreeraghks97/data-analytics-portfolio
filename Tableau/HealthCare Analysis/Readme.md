# HealthCare Analysis (Tableau)

## 📌 Overview
Dashboard analyzing hospital Length of Stay metrics by department, severity, and demographics.

## 📂 Dataset
- Source: Hospital LOS dataset
- Format: CSV/Excel with patient records

## ⚙️ Techniques Used
- Bar charts for LOS distribution
- Filters for severity and ward type
- KPIs for admissions

## 📊 Key Insights
- Emergency admissions had the longest average LOS.
- Certain departments had significantly higher patient counts.
- Bed grade and facility code influenced LOS.
