# UK Bank Analysis (Tableau)

## 📌 Overview
Dashboard analyzing customer demographics, balances, and regional distribution.

## 📂 Dataset
- Source: UK Bank dataset
- Format: CSV/Excel with customer records

## ⚙️ Techniques Used
- Pie charts for gender distribution
- Bar charts for age and job classification
- KPIs for balance levels

## 📊 Key Insights
- Majority of customers were white-collar workers.
- England had the largest customer base.
- Most customers were aged 30–40.
