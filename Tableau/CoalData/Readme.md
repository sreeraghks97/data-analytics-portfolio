# Coal Data Analysis (Tableau)

## 📌 Overview
Dashboard analyzing coal production and machine performance using moving averages.

## 📂 Dataset
- Source: Coal production dataset
- Format: CSV/Excel with machine output records

## ⚙️ Techniques Used
- Moving averages with null handling
- Line charts for machine performance
- KPIs for idle capacity

## 📊 Key Insights
- Machines RL2 and SR1 showed the highest production variability.
- Data gaps highlighted operational inefficiencies.
