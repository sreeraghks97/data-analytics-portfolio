# ⛏️ Coal Data Analysis Dashboard

An interactive Tableau dashboard developed to analyze coal production data, machine performance, and operational efficiency. The dashboard leverages time-series visualizations and moving average analysis to monitor production trends, identify performance variations, and support operational decision-making.

---

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on coal production data using Tableau. The dashboard explores machine-wise production trends, moving averages, and idle capacity to evaluate operational performance and highlight production patterns through interactive visualizations.

---

## 🎯 Objectives

- Analyze coal production across different machines.
- Monitor production trends using moving average analysis.
- Explore idle capacity and operational performance.
- Build an interactive dashboard for production monitoring.
- Present operational insights through effective data visualization.

---

## 📂 Dataset

**Dataset:** Coal Production Records

The dataset includes:

- Machine ID
- Date
- Production Output
- Idle Capacity
