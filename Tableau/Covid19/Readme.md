# Covid-19 Analysis (Tableau)

## 📌 Overview
Dashboard visualizing Covid-19 cases, deaths, and recovery rates across India.

## 📂 Dataset
- Source: Covid-19 dataset
- Format: CSV/Excel with state-level case data

## ⚙️ Techniques Used
- Tableau maps for state distribution
- Scatter plots for death vs recovery
- Pie charts for case distribution

## 📊 Key Insights
- Maharashtra and Tamil Nadu had the highest confirmed cases.
- Total confirmed cases exceeded 10 million.
- Recovery rates varied significantly across states.
