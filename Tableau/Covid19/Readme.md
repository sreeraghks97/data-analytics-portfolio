# 🦠 Covid-19 Analysis Dashboard

An interactive Tableau dashboard developed to analyze Covid-19 data across India. The dashboard visualizes confirmed cases, recoveries, deaths, and state-wise distribution to support public health analysis through interactive and data-driven visualizations.

---

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on Covid-19 data using Tableau. The dashboard explores the spread of Covid-19 across Indian states by analyzing confirmed cases, recoveries, deaths, and regional trends, enabling users to understand the pandemic through interactive visualizations.

---

## 🎯 Objectives

- Analyze confirmed cases, recoveries, and deaths across Indian states.
- Explore the geographic distribution of Covid-19 cases.
- Compare recovery and mortality trends.
- Build an interactive dashboard for pandemic data analysis.
- Present public health insights through effective data visualization.

---

## 📂 Dataset

**Dataset:** Covid-19 State-Level Records

The dataset includes:

- State Name
- Confirmed Cases
- Deaths
- Recoveries
- Date

## 📊 Dashboard Components

- **State-wise Case Distribution** – Visualize confirmed Covid-19 cases across India using geographic maps.
- **Recovery vs. Death Analysis** – Compare recovery and mortality trends using scatter plots.
- **Case Composition Analysis** – Display the distribution of confirmed, recovered, and deceased cases.
- **Interactive Filters** – Enable dynamic exploration of state-level Covid-19 data.
