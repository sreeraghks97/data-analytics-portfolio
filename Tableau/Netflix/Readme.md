# Netflix Analysis (Tableau)

## 📌 Overview
Dashboard analyzing Netflix catalog by type, release year, country, genre, and rating.

## 📂 Dataset
- Source: Netflix dataset
- Format: CSV/Excel with show and movie records

## ⚙️ Techniques Used
- Bubble charts for type distribution
- Line charts for release year trends
- Maps for country distribution

## 📊 Key Insights
- TV-MA was the most common rating.
- Content surged after 2015.
- USA and India contributed the largest share of titles.
