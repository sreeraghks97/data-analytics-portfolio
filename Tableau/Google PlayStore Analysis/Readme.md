# 📱 Google Play Store Analysis Dashboard

An interactive Tableau dashboard developed to analyze Google Play Store applications, user ratings, installs, and sentiment trends. The dashboard enables users to explore app performance across categories, user feedback, and market trends through interactive visualizations, supporting product and business analysis.

---

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on Google Play Store data using Tableau. The dashboard explores app categories, installs, ratings, and user sentiment to identify performance patterns and present meaningful insights through interactive visualizations.

---

## 🎯 Objectives

- Analyze app installs across different categories.
- Explore ratings and download trends.
- Compare user sentiment across applications.
- Build an interactive dashboard for app performance analysis.
- Present market insights using effective data visualization.

---

## 📂 Dataset

**Dataset:** Google Play Store Records

The dataset includes:

- App Name
- Category
- Rating
- Number of Installs
- User Reviews
- Sentiment Polarity
