# Google PlayStore Analysis (Tableau)

## 📌 Overview
Dashboard analyzing app categories, ratings, and user sentiments.

## 📂 Dataset
- Source: Google Play Store dataset
- Format: CSV/Excel with app details and reviews

## ⚙️ Techniques Used
- Sentiment analysis visualization
- Bar charts for category installs
- KPIs for ratings and downloads

## 📊 Key Insights
- Positive sentiment dominated reviews.
- Productivity and gaming apps led installs.
- Some apps had strong negative sentiment polarity.
