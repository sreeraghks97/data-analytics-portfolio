# 🏠 Airbnb Analysis Dashboard

An interactive Tableau dashboard developed to analyze Airbnb listing data, pricing trends, occupancy patterns, and property distribution. The dashboard enables users to explore market performance across different locations and property types through interactive visualizations, supporting data-driven business analysis.

---

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on Airbnb listing data using Tableau. The dashboard explores pricing, occupancy, property types, and geographic distribution to identify market trends and provide insights through interactive visualizations.

---

## 🎯 Objectives

- Analyze pricing variations across property types and locations.
- Explore occupancy trends and seasonal demand patterns.
- Compare property performance across different accommodation types.
- Build an interactive dashboard for market analysis.
- Present hospitality insights using effective data visualization.

---

## 📂 Dataset

**Dataset:** Airbnb Listings

The dataset includes:

- Property ID
- Location
- Property Type
- Price
- Occupancy Rate
- Availability

## 📊 Dashboard Components

- **Pricing Analysis** – Compare average prices across property types and locations.
- **Occupancy Trend Analysis** – Explore occupancy patterns over different time periods.
- **Property Type Comparison** – Analyze pricing and occupancy across accommodation types.
- **Geographic Distribution** – Visualize Airbnb listings across different locations using maps.
- **Interactive Filters** – Enable dynamic exploration by location and property type.
