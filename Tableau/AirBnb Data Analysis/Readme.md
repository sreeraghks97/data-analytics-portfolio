# Airbnb Analysis (Tableau)

## 📌 Overview
Interactive Tableau dashboard analyzing Airbnb listings, pricing, and occupancy trends.

## 📂 Dataset
- Source: Airbnb dataset
- Format: CSV/Excel with property details, pricing, and location

## ⚙️ Techniques Used
- Tableau data connections
- Filters for location and property type
- KPIs for average price and occupancy
- Charts: line, bar, map

## 📊 Key Insights
- Pricing varied significantly by location and property type.
- Occupancy rates peaked during holiday seasons.
- Entire homes/apartments had higher average prices compared to shared rooms.
