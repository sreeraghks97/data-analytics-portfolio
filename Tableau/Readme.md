# Tableau Projects

This folder contains interactive dashboards built in Tableau.  
Each project demonstrates skills in data visualization, filters, KPIs, and storytelling.

## 📂 Projects

### [Airbnb Analysis](AirBnb%20Data%20Analysis/)
- Dashboard analyzing Airbnb listings, pricing, and occupancy trends.
- Key insight: Pricing varied significantly by location and property type.
![Airbnb Dashboard](AirBnb%20Data%20Analysis/AirbnbAnalysis%20Dashboard.jpg)

---

### [Air Travel Analysis](AirTravelAnalysis/)
- Dashboard visualizing airline performance and passenger trends.
- Key insight: Seasonal peaks in travel demand; certain airlines dominated specific routes.
![Air Travel Dashboard](AirTravelAnalysis/AirTravelAnalysis%20Dashboard.jpg)

---

### [Coal Data Analysis](CoalData/)
- Dashboard analyzing coal production and machine performance using moving averages.
- Key insight: Machines RL2 and SR1 showed the highest production variability.
![Coal Dashboard](CoalData/CoalData%20Dashboard.jpg)

---

### [Covid-19 Analysis](Covid19/)
- Dashboard visualizing Covid-19 cases, deaths, and recovery rates across India.
- Key insight: Maharashtra and Tamil Nadu had the highest confirmed cases.
![Covid-19 Dashboard](Covid19/Covid19%20Dashboard.jpg)

---

### [Google PlayStore Analysis](Google%20PlayStore%20Analysis/)
- Dashboard analyzing app categories, ratings, and user sentiments.
- Key insight: Positive sentiment dominated reviews; productivity and gaming apps led installs.
![PlayStore Dashboard](Google%20PlayStore%20Analysis/GooglePlayStore%20Dashboard.jpg)

---

### [Hospital LOS Analysis](HealthCare%20Analysis/)
- Dashboard analyzing hospital Length of Stay metrics by department, severity, and demographics.
- Key insight: Emergency admissions had the longest average LOS.
![Hospital LOS Dashboard](HealthCare%20Analysis/HealthCareAnalysis/%20Dashboard.jpg)

---

### [House Data Analytics](House%20Data%20Analytics/)
- Dashboard analyzing house prices against crime rate, pollution, distance, and rooms.
- Key insight: Crime rate and pollution had strong negative correlation with house prices.
![House Dashboard](House%20Data%20Analytics/HousingData%20Dashboard.jpg)

### [HR Analysis](HR%20Analysis/)
- Dashboard analyzing employee attrition, demographics, and job roles.
- Key insight: Sales department had the highest attrition; younger employees more likely to leave.
![HR Dashboard](HR%20Analysis/HR_Analysis%20Dashboard.jpg)

### [Netflix Analysis](Netflix%20Analysis/)

### [Rainfall Analysis](Rainfall%20Analysis/)
