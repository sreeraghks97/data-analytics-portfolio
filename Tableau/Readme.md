# 📊 Tableau Data Analytics Projects

This folder contains interactive Tableau dashboards developed to analyze real-world datasets across multiple domains including business, healthcare, finance, HR, entertainment, and environmental analysis.

Each project demonstrates skills in:
- Data visualization and dashboard development
- KPI creation and performance tracking
- Interactive filters and user-driven exploration
- Trend analysis and pattern identification
- Business insight generation through data storytelling

The objective of these projects is not only to visualize data but also to transform raw information into meaningful insights that support data-driven decision-making.

---

# 📂 Projects

## 🏠 [Airbnb Analysis](AirBnb%20Data%20Analysis/)

**Domain:** Hospitality & Real Estate Analytics

Analyzed Airbnb listings to understand pricing patterns, property distribution, and occupancy trends across locations.

**Key Insight:**
- Pricing varied significantly based on location and property characteristics, highlighting the impact of market demand on listing prices.

![Airbnb Dashboard](AirBnb%20Data%20Analysis/AirbnbAnalysis%20Dashboard.jpg)
