# 🏡 House Data Analytics Dashboard

An interactive Tableau dashboard developed to analyze housing data and explore the relationship between property prices and key factors such as crime rate, pollution levels, distance from the city center, and number of rooms. The dashboard enables users to identify market trends and supports data-driven real estate analysis through interactive visualizations.

---

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on housing data using Tableau. The dashboard explores the factors influencing property prices by analyzing environmental, geographical, and structural attributes, presenting insights through interactive visualizations.

---

## 🎯 Objectives

- Analyze the relationship between crime rate and house prices.
- Explore the impact of pollution levels on property values.
- Evaluate the influence of distance from the city center on housing prices.
- Compare property values based on the number of rooms.
- Build an interactive dashboard for real estate market analysis.

---

**Dataset:** Housing Records

The dataset includes:

## 📊 Dashboard Components

- **Crime Rate Analysis** – Explore the relationship between crime rate and house prices.
