# 🛒 SuperMarket Sales Dashboard | Microsoft Excel

![Dashboard Preview](SuperMarket-Sales-Dashboard.png)

## 📌 Project Overview

This Excel dashboard analyzes supermarket sales data to evaluate revenue, gross income, product category performance, city-wise sales, and customer payment preferences. The interactive dashboard enables users to explore sales trends and identify key business insights through dynamic visualizations.

---

## 🎯 Business Objective

The objective of this project is to analyze supermarket sales data to understand revenue distribution, product performance, customer purchasing behavior, and payment preferences. The dashboard provides an interactive platform for exploring sales performance and supporting data-driven business decisions.

---

## 📂 Dataset

**Source:** SuperMarket Sales Dataset

The dataset includes information such as:

- Product Line
- City
- Sales Revenue
- Gross Income
- Payment Method
- Customer Type
- Transaction Details

---

## 📊 Dashboard Features

- Interactive dashboard built using Pivot Tables and Pivot Charts
- Product category performance analysis
- City-wise revenue comparison
- Payment method distribution
- Gross income analysis
- Interactive filtering for dynamic data exploration

---

## 🛠️ Tools & Skills Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Dashboard Design
- Data Visualization

---

## 🔍 Key Findings

- Health & Beauty products generated the highest gross income.
- Naypyitaw contributed the highest overall sales revenue.
- E-wallet was the most frequently used payment method.
- Interactive filtering enables users to compare sales performance across product categories, cities, and payment methods.

---

## ✅ Conclusion

This dashboard transforms supermarket sales data into an interactive business reporting solution that highlights revenue distribution, product performance, and customer purchasing patterns. It demonstrates the effective use of Microsoft Excel for dashboard development, Pivot Table analysis, conditional formatting, and interactive data visualization.
