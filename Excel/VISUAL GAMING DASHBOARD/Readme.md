# 🎮 Gaming Dashboard | Microsoft Excel

![Dashboard Preview](Visual-Gaming-Dashboard.png)

## 📌 Project Overview

This Excel dashboard analyzes gaming data across Indian states to evaluate user activity, revenue generation, and regional performance. The interactive dashboard provides insights into state-wise revenue distribution and user engagement through dynamic visualizations.

---

## 🎯 Business Objective

The objective of this project is to analyze gaming data to understand revenue trends, user activity, and regional performance across different states. The dashboard enables users to compare key metrics and identify high-performing regions through interactive reporting.

---

## 📂 Dataset

**Source:** Gaming Dataset

The dataset includes information such as:

- State
- Revenue
- Active Users
- Average Daily Users
- Gaming Activity Metrics

---

## 📊 Dashboard Features

- Interactive dashboard built using Pivot Tables and Pivot Charts
- State-wise revenue analysis
- User activity and engagement analysis
- Average daily user trend visualization
- Interactive filtering for dynamic data exploration

---

## 🛠️ Tools & Skills Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Data Visualization
- Data Analysis

---

## 🔍 Key Findings

- Bihar generated the highest revenue among all states in the dataset.
- Average daily active users peaked between **900–1000**.
- Karnataka and Tamil Nadu contributed significantly to overall revenue.
- Interactive filtering enables users to compare revenue and user activity across different states.

---

## ✅ Conclusion

This dashboard transforms gaming data into an interactive analytical solution that highlights regional revenue and user engagement trends. It demonstrates the effective use of Microsoft Excel for dashboard development, Pivot Table analysis, and interactive reporting while enabling users to explore gaming performance across multiple states.
