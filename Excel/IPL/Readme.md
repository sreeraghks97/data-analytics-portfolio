# 🏏 IPL Dashboard | Microsoft Excel

![Dashboard Preview](IPL-Dashboard.png)

## 📌 Project Overview

This Excel dashboard analyzes Indian Premier League (IPL) match data from 2008–2017 to provide insights into team performance, player achievements, match venues, and season-wise trends. Interactive filters enable users to explore match statistics across different seasons, cities, venues, teams, and award-winning players.

---

## 🎯 Business Objective

The objective of this project is to analyze IPL match data to evaluate team performance, identify top-performing players, and explore historical match trends. The dashboard provides an interactive platform for comparing teams, venues, and seasons through dynamic visualizations.

---

## 📂 Dataset

**Source:** IPL Match Dataset

The dataset includes information such as:

- Season
- Teams
- Match Winner
- Toss Winner
- Player of the Match
- City
- Venue
- Match Results

---

## 📊 Dashboard Features

- Interactive slicers for Season, City, Venue, Toss Winner, and Player of the Match
- Team-wise match performance analysis
- Player performance visualization
- Venue-wise match distribution
- Interactive charts for season-wise exploration

---

## 🛠️ Tools & Skills Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Data Visualization
- Data Analysis

---

## 🔍 Key Findings

- Chennai Super Kings recorded the highest number of match victories in the dataset.
- Suresh Raina and Rohit Sharma emerged as the top-performing players based on match awards.
- Kolkata and Bangalore hosted the highest number of IPL matches.
- Interactive filtering enables users to compare team performance and match statistics across different seasons and venues.

---

## ✅ Conclusion

This dashboard transforms IPL match data into an interactive analytical solution for exploring team performance, player achievements, and venue statistics. It demonstrates the effective use of Microsoft Excel for dashboard development, Pivot Table analysis, and interactive reporting through dynamic visualizations.
