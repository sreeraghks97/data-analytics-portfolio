# IPL Dashboard (Excel)

## 📌 Overview
Interactive Excel dashboard analyzing IPL matches from 2008–2017.  
Includes filters for season, city, venue, toss winner, and player of the match.

## 📂 Dataset
- Source: IPL match dataset
- Format: Excel file with match details

## ⚙️ Techniques Used
- Pivot tables for match statistics
- Slicers for season and team filters
- Charts (bar, pie) for visualization

## 📊 Key Insights
- Chennai Super Kings won the most matches.
- SK Raina and RG Sharma were top performers.
- Venues like Kolkata and Bangalore hosted the most matches.
